# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-06
### Added
- Initial `pinepy` package with Pine-like `series`, `ta`, `math`, `array`, and `input` namespaces.
- Core indicators (SMA, EMA, RMA, WMA, VWMA, RSI, ATR, true range, etc.).
- Pine-like history referencing via `Series.__getitem__` and `series.ref`.
- Documentation: `README.md`, `docs/USAGE.md`, `docs/PUBLISHING.md`.
